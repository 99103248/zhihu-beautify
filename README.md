# Zhihu Beautify

一个简单的 Chrome 浏览器插件，通过 CSS 和 JS 调整了部分知乎 Web PC 端的 UI 样式，优化刷知乎的体验。

## 安装

- 下载 zip 或者 git clone 获得项目代码
- Chrome 设置 -> 扩展程序 -> 开启开发者模式 -> 加载已解压的扩展程序 -> 选择当前项目的目录
